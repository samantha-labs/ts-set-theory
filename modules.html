areSetsDisjoint

  • areSetsDisjoint<TElement>(a: Set<TElement>, b: Set<TElement>): boolean
    • Defined in predicates.ts:19

    Checks if two sets are disjoint; that is, if both sets have no elements in common.

    • Formal definition: A ∩ B = ϕ
    • This method runs in bilinear time, or O(m * n), where m = |B| (size of set B) and n = |A| (size of set A).
    example
    // {1, 2, 3} ∩ {4, 5, 6} = ϕ
    areSetsDisjoint(new Set([1, 2, 3]), new Set([4, 5, 6]); // true

    // {1, 2, 3} ∩ {3, 4, 5} ≠ ϕ
    areSetsDisjoint(new Set([1, 2, 3]), new Set([3, 4, 5]); // false

areSetsEqual

  • areSetsEqual<TElement>(a: Set<TElement>, b: Set<TElement>): boolean
    • Defined in predicates.ts:51

    Checks if two sets are equal, where two sets have the same number of elements, and same elements themselves

    • Formal definition: A = B means ∀z, (z ∈ x ⇔ z ∈ y)
    • This method runs in linear time, or O(n), where n = |A| (size of set A).
    • Note: This does not check for reference/entity equality, but rather value equality.
    example
    // {1, 2, 3, 4} = {1, 2, 3, 4}
    areSetsEqual(new Set([1, 2, 3, 4]), new Set([1, 2, 3, 4])); // true

    // {hello, world} = {hello, world}
    areSetsEqual(new Set(['hello', 'world']), new Set(['hello', 'world'])); // true

    Represents the difference set operation, where the difference is the set of elements in set B that are not in set A.

    • Formal definition: B \ A = {x: x ∈ B | x ∉ A}
    • This method runs in linear time, or O(n), where n = |B| (size of set B).

getLogDice

  • getLogDice<TElement>(a: Set<TElement>, b: Set<TElement>): number
    • Defined in similarity.ts:46

    Implementation of the logDice algorithm, which was originally formulated by Pavel Rychlý. [1]

    • This method runs in logarithmic bilinear time, or O(log(n*m)), where n = |A|, and m = |B|.
    • [1] “A Lexicographer-Friendly Association Score - Sketchengine.eu” (Faculty of Informatics, Masaryk University, November 24, 2008)
    see

    https://www.sketchengine.eu/wp-content/uploads/2015/03/Lexicographer-Friendly_2008.pdf

    • b: Set<TElement>

    A coefficient score that can be interpreted into the number of collocation pairs between members of set and set B (as defined by §4 "logDice):

    • 14: The theoretical maximum score, where all occurrences of A co-occur with B and vice-versa
    • 0: There is less than 1 co-occurrence of AB for every 16,000 A or every 16,000 B
    • < 0: No statistical significance of AB collocation
    • The score is independent of the given corpus's size
